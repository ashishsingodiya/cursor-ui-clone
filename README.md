# Cursor UI Clone

A static UI clone of the Cursor landing page, built with plain HTML and CSS.

## Preview

Open `index.html` in your browser.

## Live Demo

https://work.ashish.pro/cursor-ui-clone

## Project Structure

- `index.html` - Main markup
- `style.css` - Styles
- `assets/` - Images, fonts, and logos

## Notes

- No build step or dependencies.
- Desktop-only layout; not responsive for mobile.
